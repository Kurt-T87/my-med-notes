# Ribociclib-associated vortex keratopathy (2020)

> [!quote] 
> 这是临床上首次报道瑞波西利导致旋涡状角膜病变的案例。

## 临床要点
- **患者主诉：** 明显的眼部干涩感、异物感。
- **体征：** 裂隙灯检查发现角膜底部典型的螺旋状棕色色素沉着。
- **转归：** 停药后病变可逆。
